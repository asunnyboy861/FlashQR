# Pricing Configuration

## Monetization Model: Free Download + One-Time IAP Purchase

- **Price**: Free to download
- **IAP**: Non-consumable one-time purchase for Pro upgrade
- **Subscription**: None

## Free Tier (Forever Free)

- Save up to 3 QR code cards
- EDR highlight display (core feature)
- Traditional brightness adjustment display
- Import QR codes from photo library
- Camera scan import
- Manual QR code content entry
- Auto brightness restore (2 minutes)
- Manual restore button
- Dark mode
- No ads (free tier also ad-free!)

## Pro Upgrade (One-Time Purchase $1.99)

- All free tier features
- Unlimited card count
- Custom auto-restore timer (1-10 minutes)
- Home screen widgets (lock screen + home screen)
- Apple Watch app
- iCloud sync
- Custom card colors/icons
- QR code generator (generate QR from text)
- Clipboard quick import
- Card grouping/folders

## Why Not Subscription

| Consideration | Analysis |
|---------------|----------|
| Feature complexity | Minimalist tool, no ongoing development cost |
| API fees | None (all local processing) |
| Server fees | None (no backend) |
| User expectation | Utility app users strongly prefer one-time purchase |
| Competitive differentiation | Competitors almost all use subscriptions; one-time purchase is a differentiator |
| App Store trends | Apple increasingly requires subscription apps to provide ongoing value |
